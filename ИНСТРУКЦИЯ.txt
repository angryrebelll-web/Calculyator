  КАК ИСПРАВИТЬ ОШИБКУ 404 НА GITHUB PAGES
═══════════════════════════════════════════════════════════════

ПРОБЛЕМА:
GitHub Pages не находит index.html, потому что файлы в подпапке.

РЕШЕНИЕ:

1. Откройте файл INDEX_FOR_ROOT.html в этой папке
2. Скопируйте ВСЁ его содержимое (Ctrl+A, Ctrl+C)
3. На GitHub откройте ваш репозиторий: 
   https://github.com/angryrebelll-web/Calculyator
4. Нажмите кнопку "Add file" → "Create new file"
5. Назовите файл: index.html
6. Вставьте скопированное содержимое (Ctrl+V)
7. Нажмите "Commit new file" внизу страницы
8. Подождите 2-3 минуты

ВАЖНО:
- Файл index.html должен быть в КОРНЕ репозитория (не в подпапке!)
- В настройках GitHub Pages выберите "/ (root)" как папку

Проверьте настройки:
https://github.com/angryrebelll-web/Calculyator/settings/pages

После этого сайт должен работать:
https://angryrebelll-web.github.io/Calculyator/
