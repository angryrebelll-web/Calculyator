ШАГ 1: Создайте файл .nojekyll в КОРНЕ репозитория
───────────────────────────────────────────────────────────────
1. Откройте: https://github.com/angryrebelll-web/Calculyator
2. Нажмите "Add file" → "Create new file"
3. Название: .nojekyll (с точкой в начале!)
4. Оставьте файл ПУСТЫМ
5. Нажмите "Commit new file"

ШАГ 2: Создайте файл index.html в КОРНЕ репозитория
───────────────────────────────────────────────────────────────
1. Откройте файл "index_ROOT_SIMPLE.html" в этой папке
2. Скопируйте ВСЁ содержимое (Ctrl+A, Ctrl+C)
3. На GitHub: "Add file" → "Create new file"
4. Название: index.html
5. Вставьте скопированное (Ctrl+V)
6. Нажмите "Commit new file"

ШАГ 3: Проверьте настройки GitHub Pages
───────────────────────────────────────────────────────────────
1. Откройте: https://github.com/angryrebelll-web/Calculyator/settings/pages
2. Убедитесь:
   - Branch: main
   - Folder: / (root) - КОРЕНЬ
3. Сохраните

ШАГ 4: Подождите 2-3 минуты
───────────────────────────────────────────────────────────────
GitHub Pages обновляется автоматически.

Проверьте сайт:
https://angryrebelll-web.github.io/Calculyator/
